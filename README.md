# Meetup / Society
This project simulates how political opinions evolve through group interactions among individuals with different identities. When two individuals influence each other's opinions towards their own (indicated by a match score above 0.5), they are connected by an edge or line in the visualization.

## How to run
Use dotnet run or dotnet fsi in the terminal. You need .NET.
